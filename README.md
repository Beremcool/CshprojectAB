Это финальный проект по C#.
____________________________

Как запустить

```bash
cd TaskHub
dotnet run
```

После запуска появится меню. Нужно просто вводить номер пункта и следовать подсказкам в консоли.

В приложении можно:

создать задачу с названием, описанием, приоритетом, дедлайном и статусом;
посмотреть все задачи;
отдельно посмотреть выполненные, невыполненные и задачи с высоким приоритетом;
отредактировать задачу по ID;
удалить задачу;
отметить задачу как выполненную;
искать задачи по названию, статусу или приоритету;
посмотреть статистику;
сохранить задачи в файл;
загрузить задачи из файла.

Еще есть фоновая проверка дедлайнов. Она работает отдельно от основного меню и раз в 10 секунд проверяет, появились ли просроченные задачи. Если появились, программа выводит уведомление в консоль.

Код разделен по папкам:

```text
Models/        модели и enum'ы
Services/      основная логика задач, статистика и проверка дедлайнов
Repositories/  сохранение и загрузка из JSON
Utilities/     ввод и вывод в консоль
Delegates/     делегат для фильтрации задач
```

При сохранении создается файл:

```text
tasks.json
```

Он лежит рядом с запускаемым приложением. Если файла еще нет, при загрузке программа просто вернет пустой список задач.
